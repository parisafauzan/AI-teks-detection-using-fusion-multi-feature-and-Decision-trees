## 📌 Overview

This project aims to detect whether a text is written by a human or generated by an AI language model using a **Hybrid Feature Fusion** approach.

Unlike conventional AI text detectors that rely on a single feature extraction method, this research combines multiple complementary feature representations:

- ✍️ Stylometric Features
- 📊 TF-IDF Features
- 🧠 Transformer Embeddings (BERT)
- 📈 Perplexity Scores (GPT-2)

These heterogeneous features are fused into a unified representation and evaluated using several machine learning algorithms to improve robustness and generalization across unseen domains.

---

## 🎯 Research Objectives

The primary goals of this project are:

- Develop a robust AI-generated text detection system.
- Combine linguistic, statistical, semantic, and probabilistic features into a single classification pipeline.
- Evaluate the effectiveness of Hybrid Feature Fusion against domain shift problems.
- Compare the performance of:
  - Random Forest
  - XGBoost
  - CatBoost

---

## 🏗️ System Architecture

The proposed pipeline consists of several stages:

```text
Raw Text
    │
    ├── Stylometric Feature Extraction
    ├── GPT-2 Perplexity Calculation
    ├── TF-IDF Vectorization
    └── BERT Embedding Extraction
            │
            ▼
      Dimensionality Reduction
      ├── PCA (BERT Features)
      └── TruncatedSVD (TF-IDF Features)
            │
            ▼
      Hybrid Feature Fusion
            │
            ▼
      Feature Normalization
            │
            ▼
 Machine Learning Classifiers
 ├── Random Forest
 ├── XGBoost
 └── CatBoost
            │
            ▼
         Evaluation
```

## 🔬 Feature Extraction

### 1. Stylometry

Captures writing style characteristics such as:

- Average word length
- Average sentence length
- Vocabulary diversity
- Capitalization ratio
- Punctuation ratio
- POS-tag distributions
- Lexical richness

### 2. TF-IDF

Extracts lexical patterns and term importance using:

- Unigrams
- Bigrams
- Maximum 1000 features

### 3. Transformer Embeddings

Uses:

```text
bert-base-uncased
```

to capture contextual semantic information from text.

### 4. Perplexity

Uses:

```text
GPT-2
```

to measure how predictable a text is from a language modeling perspective.

Lower perplexity often indicates AI-generated content, while human-written text tends to exhibit greater variability.

---

## 📊 Feature Fusion Strategy

Before fusion, high-dimensional features are reduced:

| Feature Type | Original Dimension | Reduction Method | Final Dimension |
|-------------|-------------------|------------------|----------------|
| Stylometry | 18 | - | 18 |
| Perplexity | 1 | - | 1 |
| TF-IDF | 1000 | TruncatedSVD | 400 |
| BERT | 768 | PCA | 300 |

Final feature vector:

```text
18 + 1 + 400 + 300 = 719 Features
```

## 🤖 Classification Models

Three machine learning algorithms are evaluated:

### Random Forest
- Ensemble tree-based classifier
- Feature importance analysis

### XGBoost
- Gradient boosting framework
- Handles high-dimensional data effectively
- Built-in regularization

### CatBoost
- Ordered boosting
- Better generalization capability
- Resistant to overfitting and domain shift

## 📈 Evaluation Metrics

The system is evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score

Validation strategy:

```text
Stratified 5-Fold Cross Validation
```

Additionally, external validation is conducted using unseen datasets to evaluate real-world generalization performance.
